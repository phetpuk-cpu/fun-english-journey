# 💎 แผนอัพเกรดความเข้มข้น — เตรียมไว้สำหรับพรีเมียมเมมเบอร์ชิป
อัปเดต: 10 ก.ค. 2026

## บริบท
Phase 0 (ตอนนี้) มี 16 บทแรก (ป.3-4 หน่วย 1-2) เนื้อหาตรงกับ สพฐ. หลักสูตรแกนกลางและจุดไวยากรณ์หลักตรงกับ Cambridge Movers (ดูสรุปเทียบหลักสูตรในบทสนทนา 10 ก.ค. 2026) แต่ยังมีช่องว่างเทียบกับสาย EP/เอกชนเข้มข้น 4 จุด:
1. ปริมาณคำศัพท์น้อยกว่า (128 คำ vs Movers wordlist ~400 คำ)
2. ไม่มีฝึกเขียน/อ่านจับใจความ (มีแต่ฟัง+เลือก+ลากคำ)
3. ไม่มีฝึกตั้งคำถามเอง (มีแต่รับรู้/เลือกคำตอบ)
4. ไม่มี Past Simple / Future tense (บางโรงเรียน EP เข้มข้นสอนคู่กับ Present Continuous ตั้งแต่ ป.4 เทอมปลาย)

แผนนี้ใช้ช่องว่างทั้ง 4 ข้อเป็นแกนออกแบบ "สิ่งที่พรีเมียมมีแต่ฟรีไม่มี" — ไม่ใช่ล็อกเนื้อหาเดิมไว้เฉยๆ แต่ทำของใหม่ที่แก้ pain point จริงของกลุ่มผู้ปกครองสาย EP

## โครงสร้าง Free vs Premium

### Free tier (ตัวดึงดูด)
- 16 บทแรกที่ทำเสร็จแล้ว (ป.3-4 หน่วย 1-2) เล่นได้ฟรีตลอดไป — ให้ลองของจริงก่อนตัดสินใจจ่าย
- 1 โปรไฟล์เด็กต่อเครื่อง, ดาว/XP พื้นฐาน, ไม่มีโฆษณา (โฆษณาในแอปเด็กไม่เหมาะ)

### Phase 2 — ขยายความกว้าง (เก็บเงินจาก "ปริมาณ")
- ทำต่อ ป.3-4 หน่วย 3-8 (อีก 48 บท ตามแผน curriculum-48-units.md เดิม)
- ขยาย ป.1-2 และ ป.5-6 ตามแผนแม่บท → ครอบคลุมทั้งประถม
- ใช้ pipeline/engine ปัจจุบันได้เลยไม่ต้องแก้โค้ด แค่เพิ่ม data/*.json + assets ต่อ — **เริ่มทำได้ทันทีตอนนี้**

### Phase 3 — เพิ่มความลึกระดับ EP (เก็บเงินจาก "ความเข้มข้น")
แก้ช่องว่าง 4 ข้อด้านบนตรงๆ:
- **โหมดเขียน** — บางจุดในแต่ละบทเปลี่ยนจาก drag-word-order เป็นพิมพ์ประโยคเอง (มี virtual keyboard + ตรวจตัวสะกดแบบยืดหยุ่น)
- **โหมดอ่านจับใจความ** — ย่อหน้าสั้น 3-4 ประโยคต่อบท + โจทย์สไตล์ Cambridge Reading & Writing (เติมคำ, จับคู่ประโยค-รูป, true/false) — มีหนังสือจริงอ้างอิงแล้ว: Oxford Skills World Reading with Writing 2 (ป.2) ให้กรอบ 6 คู่ reading-goal/writing-goal พร้อมใช้ (Read the Title/Headings → Write a Conversation, Find Key Info/Setting → Write an E-mail, Find Action Words/Main Idea → Write a Story, Find Places/Cause-Effect → Write a Paragraph, Find Likes-Dislikes/Similar-Different → Write an Opinion, Find Times/Sequence → Write a Schedule) ดู `curriculum-plan-p1-2-p5-6.md` หัวข้อ ป.2
- **โหมดตั้งคำถามเอง** — ฝึกประกอบคำถาม (What time...? / Can you...?) แทนที่จะเลือกจากตัวเลือก
- **ขยาย vocab ต่อบท** — เพิ่ม "extra vocab" เสริมนอกจาก 8 คำหลัก ให้ใกล้เคียง Cambridge wordlist มากขึ้น (opt-in ไม่บังคับ)
- **หน่วยเสริม Past Simple + Future (going to/will)** — ปลดล็อกหลัง placement quiz สั้นๆ ว่าเด็กพร้อมหรือยัง

### Phase 4 — สร้างความอุ่นใจให้ผู้ปกครอง (เก็บเงินจาก "ความมั่นใจ")
- แบบทดสอบจำลองสไตล์ Cambridge Starters/Movers/Flyers พร้อมใบประกาศนียบัตร/badge ปริ้นท์ได้
- แดชบอร์ดผู้ปกครอง: กราฟความก้าวหน้าเทียบ benchmark หลักสูตร, จุดที่ลูกติดขัดบ่อย, บทที่ควรเล่นซ้ำ
- รายงานรายสัปดาห์ (อีเมล/LINE) — ผู้ปกครองไม่ต้องเปิดแอปเองทุกวันก็ตามงานลูกได้
- ส่วน badge/certificate ทำได้เร็วเพราะ UI มีโครง result-screen (`stage.innerHTML` ใน `render()` ของ [engine.js](fun-english-journey/js/engine.js)) อยู่แล้ว ต่อยอดง่าย

### Phase 5 — ปรับให้เหมาะแต่ละคน (เก็บเงินจาก "ความสะดวก")
- หลายโปรไฟล์เด็กต่อบัญชีไม่จำกัด (พี่น้อง)
- Adaptive: ระบบเลือกบทซ้ำ/ข้ามอัตโนมัติตามคะแนนพูด+ควิซ แทนเรียงบทตายตัว
- Sync ข้ามอุปกรณ์ผ่าน cloud (ต้องมี backend — งานใหญ่สุดในแผนนี้)

## ลำดับแนะนำให้เริ่มทำจริง
1. **Phase 2 ก่อน** — ใช้ของเดิมได้เลย ต้นทุนต่ำสุด ได้ปริมาณเนื้อหาเร็วสุด
2. **Phase 4 บางส่วน** (badge/certificate) — ทำเร็ว ต่อยอดจาก UI ที่มีอยู่
3. **Phase 3** — ใช้เวลาออกแบบ UI ใหม่มากสุด แต่เป็นตัวสร้างความต่างจากแอปคู่แข่งชัดเจนที่สุด (จุดที่โรงเรียน EP เน้นแต่แอปคู่แข่งทั่วไปมักไม่มี)
4. **Phase 5 ท้ายสุด** — ต้องมี backend/cloud sync ก่อนถึงจะทำได้

## ข้อควรระวังก่อน Monetize จริง
- **PDPA**: ต้องมีหน้า consent ผู้ปกครองก่อนเก็บข้อมูลเด็ก/เก็บเงิน (ระบุไว้ใน [project-handoff.md](project-handoff.md) แล้วว่ายังไม่ทำ ต้องทำก่อน Phase 4-5)
- **License เสียง SFX**: 10 เสียงเอฟเฟกต์ตอนนี้ยืมมาจาก Unity asset pack ฟรี "Leohpaz RPG_Essentials_Free" ในโปรเจกต์ LittleVillage — ใช้แบบ private/family ความเสี่ยงต่ำ แต่ถ้าจะขายเชิงพาณิชย์จริง ต้องเช็ค EULA ของแพ็กให้แน่ใจ หรือเปลี่ยนไปสั่งทำ/ซื้อสิทธิ์ SFX ใหม่ให้ชัดเจนก่อนเปิดขาย
- **ตั้งราคาเทียบกับอะไร**: ผู้ปกครองสาย EP เทียบราคากับค่าเรียนพิเศษ/ติวเตอร์รายเดือน ไม่ใช่เทียบกับแอปเกมทั่วไป — Phase 3 (ความเข้มข้นระดับ EP) คือจุดขายที่ตั้งราคาสูงได้โดยไม่รู้สึกแพงเกินไป
