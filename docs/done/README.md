# เอกสารที่ดำเนินการเสร็จ/พิจารณาแล้ว

- `curriculum-plan-p1-2-p5-6.md`, `curriculum-reference-real-books.md`, `english-curriculum-review-thai-students.md` — แผน/รีวิวหลักสูตร ป.1-6 ซึ่งเขียนครบ 192 บทแล้ว
- `ux_ui_analysis_proposal.md` — ข้อเสนอ UX/UI ที่ implement ไปแล้ว (parents dashboard, claymorphism) มี implementation note ท้ายไฟล์
- `audio-microphone-settings-handoff-for-claude.md` — ปัญหาเสียง/ไมค์ที่ Codex แก้ไปแล้ว (AudioManager, settings screen)
- `webapp-ux-ui-security-review-for-claude.md` — รีวิว UX/security ที่ส่วนใหญ่ addressed แล้ว (XSS fix, responsive layout) ข้อที่ยังไม่ทำ (automated test) ถูกยกไปเป็นข้อเสนอแนะแยกต่างหาก
- `security_audit_plan.md` — แผน security 10 ข้อ (SEC-01 ถึง SEC-10) **ทำเฉพาะ SEC-01 (stored XSS)** ส่วนที่เหลือ **ตัดสินใจ defer** เพราะเกินความจำเป็นของแอป local ไม่มี backend
